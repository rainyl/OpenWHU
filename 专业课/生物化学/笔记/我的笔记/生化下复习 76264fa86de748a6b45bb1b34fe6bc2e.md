# 生化下复习

学习代谢途径时应该掌握以下内容

1. 代谢途径的场所和功能
2. 进入和离开代谢途径的代谢物的名称（或者化学式）
3. 如何与其他代谢途径整合协调
4. 限速步骤，限速酶的名称和调节方式
5. 调节限速酶的别构效应物的名称
6. 生成ATP或等价物的反应
7. 关键酶缺失的后果
8. 代谢途径的抑制剂

生化反应调节的位点往往是不可逆的限速步. 原因在于, 如果想要调控一个通路, 那么最好只调节这一个通路而不干涉其他反应, 因此选择不可逆反应, 而如果该不可逆反应不是限速步, 那么调节也没什么作用, 因此既要不可逆, 又要限速
